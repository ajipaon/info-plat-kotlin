# 🚗 Daftar Plat Nomor Kendaraan Indonesia Berdasarkan Provinsi

## 📖 Cara Membaca Plat Nomor

```
[ KODE WILAYAH ] [ NOMOR URUT ] [ HURUF SERI ]

Contoh:  B  1234  ABC
         ↑    ↑     ↑
      Jakarta  No.  Seri
               Urut
```

### Huruf Seri Belakang
- Huruf seri belakang umumnya menunjukkan **sub-wilayah atau periode pendaftaran**.
- Semakin banyak huruf di belakang, menandakan kendaraan lebih baru.
